<h1 align="left" style="margin-bottom: 20px; font-weight: 500; font-size: 50px; color: black;">
  FastAPI JWT Auth
</h1>


1. Install the requirements:

    ```sh
    poetry install -v
    ```

1. Init python environment:
    ```sh
    poetry shell
    ```

1. Run the app:

    ```sh
    python main.py
    ```

1. Test at [http://localhost:8080/docs](http://localhost:8080/docs)
